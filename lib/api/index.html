linkbot

A Discord bot that quietly cleans up the links your friends share.

{{if .InviteURL}} Add to Discord {{end}}

What it does

  • Universal music links. Spotify, Apple Music, YouTube Music, Tidal, and Deezer URLs are rewritten through Odesli to a song.link page anyone in the channel can open, no matter which streaming service they use.
  • No more tracking junk. Strips utm_*, fbclid, gclid, and friends so you share the actual link, not the ad-tagged one.
  • Paywalls, opened. Articles from sites like the WSJ, FT, Bloomberg, the Washington Post, and the New Yorker are rewritten through a random archive.today mirror so everyone can read along.

How to use it

Once linkbot is in your server, it watches the channels it can read. When it spots a music link or a paywalled article, it replies with a clean version. You can also run /sanitize on any URL to get a cleaned version on demand.

linkbot stays quiet if the clean link is already in the message or was shared recently in the channel, so it won’t double up on other bots’ work.
